General Instructions

Clone the Repository as such. Altering the path might affect the working of the application.
Replace API_KEY variable with OpenAI API key in the "key.py" file as a string before trying to execute the "app.py" file.

How to Execute

Open a new terminal with the appropriate path and run the command "python app.py". This runs the FLASK application in the default server URL "http://127.0.0.1:5000".
For the First time execution alone, a model of Whisper having size of around 1.2Gb will get downloaded after which the application can be run any number of times.
